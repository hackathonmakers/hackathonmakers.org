Setup
=========

### Node.Js - /server

### Como empezar?

* Instala todas las librerias. Utilizando el comando

```bash
$ npm install
```
 en la carpeta principal del repositorio.  


* Inicia el servidor nodejs utilizando el comando 'node app.js'.

```bash
$  node app.js
```

* Abri tu navegador en 'http://localhost:3000'
